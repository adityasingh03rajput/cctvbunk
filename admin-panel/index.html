ℹ️ Manual Marking Instructions

  • Select semester, branch, date, and period to load students
  • Marking a student present will mark them present for the selected period and all future periods
  • Marking a student absent will mark only the selected period as absent
  • All manual markings are logged in the audit trail
  • You can optionally add a reason for the marking
